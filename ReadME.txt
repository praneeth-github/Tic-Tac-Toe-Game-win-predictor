This file contains information regarding what the other folders contain

* The mini-project folder is the main xilinx project of our group.
* The Reports folder contains various detailed reports like the design summary, power consumption, timing delays, a few files showing schematic design of the circuit etc. They are generated in xilinx
* The code folder contains two verilog files, namely tic_tac_toe.v and tic_tac_toe_tb.v which have the verilog code for the circuit and the test bench respectively (This folder is created mainly for easier browsing of the code files. These two files are the same as those present in the main mini-project folder)
* The report contains information regarding our project members, problem statement, our design/implementation approach, simulation results etc
